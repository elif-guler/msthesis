* Analysis of strains and their parents

* TaxID and Parent TaxID Analysis

for each strain in the newlist.txt, the corresponding taxid
was retrieved using the taxi. the parent taxid is then obtained from the
taxon table in neidb. the number of genomes associated with both the strain
and its parent was determined using the supplied sql query.

#+begin_src bash
  echo -e "strain\ttaxid\tnumber_of_taxid\tparent_taxid\tnumber_of_parent_taxid"

while read line
do
    code=$(echo "$line" | cut -f1)

    strain_name=$(echo "$line" | cut -f3)

    taxid=$(taxi -e "$strain_name" neidb | tail -n 1 | awk '{print $1}')

    parent=$(sqlite3 neidb "select parent from taxon where taxid=$taxid")

    n_tax=$(sqlite3 neidb "select count(accession) from genome where taxid=$taxid and level='complete' or level='\
chromosome'")

    n_parent=$(sqlite3 neidb "select count(accession) from genome where taxid=$parent and level='complete' or lev\
el='chromosome'")

    echo -e "$code\t$taxid\t$n_tax\t$parent\t$n_parent"

done < newlist.txt
#+end_src
