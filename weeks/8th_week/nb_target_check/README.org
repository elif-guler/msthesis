#+TITLE: nb_target_check

Checks whether the target/neighbor genome labels used in Seqwin's
supplementary dataset actually match what the genomes look like
phylogenetically.

* Datasets

- clo :: C. difficile
- mtb :: M. tuberculosis
- sen :: S. enterica

Each dataset folder has an =acc.txt= table with columns:

#+begin_example
GenBank accession    Is target    Is sampled (100 genomes)    Is sampled (1000 genomes)
#+end_example

* Scripts

** runpipeline.sh <table.tsv> <outdir> [100|1000]

Downloads the genomes for the given table and sample pool (100 or 1000),
turns each one into a single FASTA file, and runs =phylonium= to get a
distance matrix. Genomes phylonium can't compare (too little homology to
the rest of the set) get dropped automatically.

** check_separation.sh <table.tsv> <outdir> [100|1000]

Uses the FASTA files from =runpipeline.sh= to build a tree (=nj= +
=midRoot= + =land=), finds the best-supported target clade (=fintac= +
=pickle=), and compares it against the labels in the table. Prints out
any genome where the label disagrees with the tree:

- labeled /target/ but the tree puts it with the neighbors
- labeled /neighbor/ but the tree puts it with the targets

** run_all.sh [100|1000] [dataset ...]

Runs both scripts above for all three datasets (or the ones you list).
Results go in =<dataset>/pool_100/= or =<dataset>/pool_1000/=.

* Usage

#+begin_src bash
chmod +x *.sh
bash run_all.sh 100     # check against the 100-genome sample
bash run_all.sh 1000    # check against the 1000-genome sample
#+end_src

* Output (per dataset, per pool)

- =first_targets.txt= / =first_neighbors.txt= :: labels from the table
- =second_targets.txt= / =second_neighbors.txt= :: labels from the tree
- =target_to_neighbor.txt= :: labeled target, tree disagrees
- =neighbor_to_target.txt= :: labeled neighbor, tree disagrees
- =all.nwk= :: the tree itself

Downloaded genomes (=genomes/=, =fastas/=, =all/=) are not kept in this repo
(see =.gitignore=) since they're large and can be regenerated by rerunning
=runpipeline.sh=.
